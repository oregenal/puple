# Puple

## Console audio player in pure C.

Simple project to explore PulseAudio.

## Details

Currently playing most 8, 16, 24 bit WAV files.
Also you can specify dir name, to play all WAV files in it.
MP3 support ahead.

### Usage

```bash
# ./puple file.wav
```
To play all files in dir:

```bash
# ./puple dir_name/
```
